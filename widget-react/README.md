# Qwik widget counter

## Prerequisites

node v22
pnpm v10

## Dev mode

```
pnpm dev
```

## Prod mode

```
pnpm build.client
pnpm build.server
pnpm build.csr
pnpm serve
```

## Prod watch mode 

NOTE: run each command in separate terminal tabs

```
pnpm build.client --watch
pnpm build.server --watch
pnpm build.csr --watch
pnpm serve:watch
```
