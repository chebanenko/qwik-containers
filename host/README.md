# Host app

## Prerequisites

node v22
pnpm v10

## Dev mode

```
pnpm run dev
```

## Prod mode

```
pnpm run build.client
pnpm run build.server
pnpm run serve
```

## Prod watch mode 

NOTE: run each command in separate terminal tabs

```
pnpm build.client --watch
pnpm build.server --watch
pnpm serve:watch
```
